# tt-motor_calibrator_for_Arduino
Для прямолинейного движения роверов, лишенных одометрии, необходимо тщательно подобрать пару моторов, характеризующуюся идентичными параметрами.

Тахометр-одометр с автоматическим отключением

Устройство предназначено для замера скорости вращения (импульсов) мотора за фиксированный интервал времени. Принцип работы:

Запуск: По нажатию кнопки (D2) включается реле (D4), подавая питание от аккумулятора 2S (7.4В) на измеряемый мотор, и одновременно начинается отсчёт времени и подсчёт импульсов.

Измерение: На валу мотора установлено колесо одометра с 20 отверстиями. Оптический датчик с формирователем на 74HC14 фиксирует прохождение отверстий и выдаёт импульсы на вход D12 Arduino. Каждый фронт импульса увеличивает счётчик.

Индикация: Текущее количество насчитанных импульсов отображается на 4-разрядном 7-сегментном дисплее TM1637. Незначащие нули спереди гасятся.

Завершение: Через заданное время (по коду — 40 секунд) реле автоматически отключает мотор. Подсчёт импульсов прекращается. На дисплее остаётся зафиксированный результат.

Питание: Вся схема — Arduino Nano, датчик, дисплей, реле — питается от встроенного стабилизатора Nano (на VIN подаётся напряжение батареи 2S). Мотор запитан от той же батареи через силовые контакты реле. Также есть возможность питания от USB.
